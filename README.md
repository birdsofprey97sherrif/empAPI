# EMPAPI

An Express.js API for managing employee data.

## 🚀 Features

- Create, read, update, and delete employee records
- File uploads
- Middleware for authentication and error handling
- MongoDB integration

## 📦 Technologies Used

- Node.js
- Express.js
- MongoDB (Mongoose)
- Multer (for file uploads)
- Dotenv
